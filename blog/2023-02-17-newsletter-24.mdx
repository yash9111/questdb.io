## üèÜ New release!

We have released QuestDB 6.7. The release includes the following main features:

- The
  [UUID data type](https://questdb.io/blog/uuid-coordination-free-unique-keys/)
  for improved query performance
- The
  [EXPLAIN](https://questdb.io/blog/explain-sql-query-plan/)
  keyword for the SQL execution plan
- [Trigonometric functions](https://questdb.io/docs/reference/function/trigonometric/)

We would like to thank our community for the contributions üíú

QuestDB 7.0 with WAL is already in beta and will be released for prod workloads
soon! We now offer a [two-week trial](https://questdb.io/cloud/) for QuestDB
Cloud.

## üêç Pandas integration

QuestDB's Python client library now supports Pandas DataFrames! You can now
[insert 1.1M rows/s](https://www.reddit.com/r/programming/comments/10di6d9/inserting_11m_rowss_from_pandas_into_questdb_with/)
from Pandas into QuestDB. Find out more about
[the integration](https://questdb.io/docs/third-party-tools/pandas/).

![Image of a QuestDB logo and a Pandas logo](/img/blog/2023-02-17/questdb-pandas.png)
