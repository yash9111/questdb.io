## The Role

As a UI / Front End Developer at QuestDB you will be an integral part of a small tight-knit team, focusing on designing and developing user interfaces. You will write simple high quality code, which is scalable and maintainable for the long run. You will fully own features and design changes.

Front-end landscape is ever changing. Therefore, we value programming experience and the "itch" to deliver value, over knowledge of some specific framework.
We expect candidates to be comfortable with the current tools we use: Typescript, React, NextJS, Docusaurus, Jest, Cypress, Github Actions.

We are looking for an individual who is passionate about open source and with a creative mindset and intellectual curiosity.

## Responsibilities 

- Design, build and maintain all QuestDB user interfaces.
- Define, design, and implement user experience for QuestDBâ€™s open source database, QuestDB Cloud and the website.
- Create storyboards, mockups, prototypes, and flows to convey QuestDB use cases and product features.
- Improve and maintain tested and automated releases.
- Ensure compatibility with back end APIs.
- Engage with community to promote new features and gather feedback.

## Requirements

- 3+ years proven track record of building web front end software.
- Expert knowledge of Typescript, HTML and CSS.
- Good grasp of popular front-end code build tools.
- A strong opinion about UX best practices.
- Proficiency in translating user feedback into product design.
- Excellent communication skills.
- Strong team player open to guiding colleagues on your areas of expertise.

### Nice to have

Having any of the following skills would make a candidate stand out:

- Prior experience with building long lasting products with scalability in mind
- Knowledge of cloud-native technologies
- Public contributions to open source projects
